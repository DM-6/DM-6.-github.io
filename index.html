4.23日笔记 svg 动画

  • yarn 代替 npm
  • yarn：facebook的包管理器，npm不好用

终端命令：

  • yarn global add browser-sync 全局安装browser-sync
  • yarn global bin 配置环境变量，将下面路径复制新建到path变量里 关闭vscode，以管理员身份运行打开
  • browser-sync start –server “./“ –files “*.html” 启动一个服务器在当前目录下，监听文件修改
    • Local: http://localhost:3000 本机调试地址
    • External: http://192.168.43.202:3000 局域网内网地址

Thank you !
